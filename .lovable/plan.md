

# Vertex Web Solutions — Premium Swiss Digital Agency Website

## Overview
A modern, high-converting single-page website for a Swiss-based digital agency. Dark navy primary color scheme, minimalist design, smooth animations, and a professional corporate look that conveys Swiss precision and quality.

## Design System
- **Primary color**: Dark navy blue
- **Accent**: Bright blue for CTAs and highlights
- **Typography**: Clean, modern sans-serif with strong hierarchy
- **Style**: Minimalist, spacious, premium feel with subtle animations (fade-in on scroll, hover effects)
- **Dark mode support** built in
